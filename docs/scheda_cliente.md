# Scheda Cliente avanzata

La scheda cliente è la **memoria di tutto** ciò che riguarda un cliente: dati, storico, situazione economica e note. Impari a usarla in pochi minuti seguendo i passaggi qui sotto.

## Passo 1 — Apri la scheda del cliente

Il modo più rapido: dall'**agenda** clicca sull'appuntamento del cliente. Si apre il pannello **Gestione Appuntamento** con il riquadro **Info Cliente**.

![Appuntamento con il riquadro Info Cliente](assets/images/scheda_cliente/step_01.png)

> In alternativa: **Anagrafiche → Clienti → [scegli il cliente] → Modifica**.

## Passo 2 — Muoviti tra le schede (tab)

La scheda è divisa in **tab**: ogni tab è una parte della storia del cliente.

![Scheda cliente completa con i tab](assets/images/scheda_cliente/step_02.png)

| Tab | Cosa trovi |
|-----|-----------|
| **Dati Anagrafici** | Nome, contatti, data di nascita, foto |
| **Appuntamenti** | Storico visite con trattamento, operatore, importo |
| **Schede Tecniche** | Formule e parametri dei servizi |
| **Abbonamenti / Prepagate** | Pacchetti e credito attivi |
| **Fidelity Card** | Punti accumulati |
| **Note** | Appunti interni, visibili solo allo staff |
| **Galleria** | Foto e allegati (es. consenso GDPR) |

## Passo 3 — Scrivi una nota interna

Apri il tab **Note** e scrivi l'informazione utile allo staff (preferenze, avvertenze). Non è visibile al cliente.

![Il tab Note della scheda cliente](assets/images/scheda_cliente/step_03.png)

![Nota interna compilata](assets/images/scheda_cliente/step_04.png)

!!! example "Esempi di note utili"
    *"Preferisce il mattino"* · *"Allergia al nichel"* · *"Cliente fedele dal 2024"*.

## Passo 4 — Ritrova la nota durante il lavoro

Una volta salvata, la nota ricompare quando apri l'appuntamento: lo staff la vede senza cercarla.

![Nota visibile nel pannello appuntamento](assets/images/scheda_cliente/step_05.png)

!!! info "Le icone in agenda e in cassa"
    La scheda accende anche le icone-promemoria: **$ rosso** = sospeso, **arancione** = prepagata, **verde** = abbonamento, **torta** = compleanno. Le vedi sia in agenda sia in cassa.

## Import massivo dei clienti (migrazione)

Se arrivi da un altro gestionale, non reinserire i clienti a mano:

1. Vai su **Anagrafiche → Import → Clienti**.
2. Prepara un file Excel/CSV con almeno **nome, cognome, telefono** (email, data di nascita e note sono facoltative).
3. Carica il file: il sistema mostra un'**anteprima**; controlla che le colonne siano abbinate correttamente e conferma.

!!! tip "Argomento che convince"
    *"Non devi reinserire 3.000 clienti a mano."* L'import massivo è spesso ciò che sblocca i saloni indecisi.

## Tag e bandiere cliente

Dalla scheda puoi aggiungere **tag** per segmentare la clientela (es. *VIP*, *top spender*, *preferisce mattino*). Serviranno per le campagne di [Marketing Automation](marketing_automation.md).
